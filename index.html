Welcome

Hi my name is Prateek. Natural Language Query Interface is a Google Summer of Code 2016 project under vitrivr.org.

Description

In this project, I have extended the vitrivr user interface to support voice queries by converting speech query input to text and thus build a natural language query interface that gives full control to the user to interact with the user interface by speaking voice commands in a complete natural way.

Main features

  • Flexible voice commands

  • Natural interaction using follow-up voice commands

  • Spoken feedback by system for natural interaction

  • Speech and mouse interaction

Short demo video of my work

vitrivr is an innovative video retrieval engine that offers a rich set of different query paradigms to search video sequences in large video collections. This video shows the natural interaction of a user with the UI of the system using natural language query interface.
